# homework 1

## the question

請建立 C 程式計算下列運算式的值，如下所示：

1. $2x^2 - 4x + 1$
2. $a^2 + b$
3. $3y^2 + 8y + 4$

## input

輸入只有一行，輸入四個以空格分隔的整數，分別代表 $x$ 、 $a$ 、 $b$、 $y$ 的值。

## output

請輸出運算後的值，並以三行分行顯示。
