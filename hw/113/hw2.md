# 作業 2

## 說明

目前商店正在週年慶折扣，消費者消費滿 1000 元(含)，就有 8 折的折扣。

請建立 C 程式計算消費者實際應付款金額。

## 輸入

輸入只有一行，輸入一個整數代表消費者消費的金額。

## 輸出

輸出消費者實際應付款金額。

## 測資

<details>
<summary>測試範例一</summary>
  
#### 輸入範例
  
```
900
```
  
#### 輸出範例
  
```
900
``` 
  
---
  
</details>
